Ionic

Ionic es un SDK (kit de desarrollo de software) de código abierto para desarrollo de aplicaciones móviles híbridas. Originalmente fue construido usando AngularJS y Apache Cordova. Sin embargo, ahora puede ser utilizado junto con cualquier framework de interfaz o incluso también sin ningún framework.

Ionic provee los servicios para desarrollar aplicaciones web, móviles y de escritorio utilizando tecnologías web como CSS, HTML5, y Sass.

Algunas ventajas

La principal ventaja de este framework es que nos permite crear aplicaciones híbridas. El desarrollo y mantenimiento de una sola aplicación con un único código para distintas plataformas resulta más rapido y menos costoso que el desarrollo de una aplicación nativa.

Además, este framework es un proyecto de código abierto, muy bien documentado y con una comunidad activa, por lo que facilita la curva de aprendizaje.

Aplicaciones que confían en Ionic

Market Watch

Market Watch es una aplicación para dispositivos móviles que brinda a los usuarios noticias de finanzas, negocios y bolsa. Además, muestra las noticias más actuales de la empresa, cotizaciones bursátiles, asesoramiento financiero y datos de mercado. Lo más importante es que, como aplicación Ionic, funciona muy bien tanto en dispositivos Android como iOS.

McDonald's (TK)

Como todas las grandes empresas, McDonald's también dedica muchos esfuerzos y recursos a crear una presencia digital sólida. Es por ello que decidieron confiar en Ionic para su aplicación móvil de pedidos de alimentos en Turquía. La aplicación ofrece distintas ofertas y descuentos para los clientes.

DIESEL

La conocida marca Diesel también tiene su propia aplicación Ionic. Ayuda al equipo de merchandising a crear informes y analizar cada tienda y producto de Diesel. Además, ayuda a la marca a encontrar mejores formas de presentarse. Debido al marco Ionic, la aplicación viene con un buen rendimiento y una interfaz de usuario de alta calidad que hace que se vea y se sienta como una aplicación nativa.

Sworkit

Además, Sworkit es una aplicación de fitness utilizada por más de 25 millones de personas. Dentro de la aplicación es posible encontrar una gran diversidad de ejercicios y videos de entrenamiento. Esta aplicación ofrece la posibilidad de entrenar en cualquier lugar, sin necesidad de equipo.

Sanvello

Sanvello (anteriormente conocido como Pacifica), es una aplicación móvil que utiliza herramientas de la terapia cognitivo-conductual para ayudar a los usuarios a controlar el estrés, la ansiedad y la depresión.

JustWatch

JustWatch es una aplicación de motor de búsqueda para contenido de transmisión en línea. La aplicación ofrece los contenidos que están legalmente disponibles y son populares en el mercado en línea. Algunos de los servicios incluidos aquí son: Netflix, Amazon Prime Video, HBO Now y otros. La aplicación está disponible para dispositivos Android e iOS.
