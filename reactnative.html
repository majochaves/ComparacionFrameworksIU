React Native

React, se trata de una biblioteca de Javascript open source que sirve para construir interfaces de usuario. De ahí surge React Native, que se trata de un framework creado por Facebook para construir principalmente aplicaciones móviles (también para aplicaciones pc) utilizando JavaScript y React.

Algunas ventajas

Una de las grandes ventajas de este framework es que nos permite crear una aplicación móvil que puede ser ejecutada tanto para Android como para iOS, sin tener que usar 2 lenguajes distintos (a diferencia de usar Java/Kotling para Android y Objective-C/Swift para iOS).

Gracias a la tecnología Bridge, el JavaScript de React Native permite comunicarse con componentes nativos, por lo que es mucho más rápido y eficiente a diferencia de otras tecnologías híbridas.

Algunas aplicaciones que confían en React Native

Facebook

Facebook es un servicio de redes y medios sociales en línea estadounidense con sede en Menlo Park, California. Su sitio web fue lanzado el 4 de febrero de 2004 por Mark Zuckerberg.

Instagram

Instagram es una aplicación y red social de origen estadounidense, propiedad de Facebook, cuya función principal es poder compartir fotografías y vídeos con otros usuarios. Está disponible para dispositivos Android, iOS y Windows 10.

Discord

Discord es un servicio de mensajería instantánea freeware de chat de voz VolP, video y chat por texto; este funciona a través de servidores; este está separado en canales ya sea de texto o de voz. Discord está disponible para Microsoft Windows, MacOS, Android, iOS y Linux.

Skype

Skype es un software propietario distribuido por Microsoft, tras haber comprado la compañía homónima y que permite comunicaciones de texto, voz y vídeo sobre Internet (VoIP). Fue diseñado en 2003 por el danés Janus Friis y el sueco Niklas Zennström (también creadores de Kazaa) y desarrollada en su solución técnica por los estonios Priit Kasesalu, Ahti Heinla y Jaan Tallinn, ya que de hecho Skype nació en Tallin, Estonia.

Pinterest

Pinterest es una plataforma que permite a los usuarios crear y administrar, en tableros personales temáticos, colecciones de imágenes como eventos, intereses, aficiones y mucho más. Los usuarios pueden buscar otros pinboards, "repinchar" imágenes para sus propias colecciones. La misión de Pinterest es "conectar a todos en el mundo, a través de cosas que encuentran interesantes"

Tesla

Tesla es una empresa estadounidense con sede en Palo Alto, California, y liderada por Elon Musk, que diseña, fabrica y vende automóviles eléctricos, componentes para la propulsión de vehículos eléctricos y baterías domésticas.
